A **Movie Recommendation System** built using **Collaborative Filtering** and **Cosine Similarity** to recommend movies similar to a given movie.

---

## 🚀 Features

✨ Recommends movies based on user ratings
📊 Uses **cosine similarity** to measure similarity between movies
📁 Built using **Python and Jupyter Notebook**
🎯 Demonstrates a basic **machine learning recommendation system**

---

## ⚙️ How It Works

1️⃣ Load movie and rating datasets
2️⃣ Filter movies with enough ratings
3️⃣ Create a movie-user matrix
4️⃣ Compute **cosine similarity**
5️⃣ Recommend movies similar to the selected movie

---

## ▶️ How to Run

Install dependencies

pip install -r requirements.txt

Open the notebook

notebooks/movie_recommendation.ipynb

Run all cells to generate recommendations.
